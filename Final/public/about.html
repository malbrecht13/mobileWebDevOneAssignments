About

Medical Treatment Notebook is a customizable notebook that can be used by any health care provider! With this notebook, you can define any condition and specify the go-to treatments you would normally use for each condition. There is no longer a need to perform an extensive search online for the treatments you typically used for a specific condition. Simply store them in this treatment notebook!
